// if문과 else
int a;
if(10>5){
    a = 10;
} else {
    a = 5
}
System.out.println(a); // 결과 10

// 삼항 연산자
int b = (10 > 5) ? 10 : 5;
System.out.println(b); // 결과 10
조건문 ? 참일때 : 거짓일때;
쉽게 얘기 하자면 (조건문) ? 참 : 거짓; 이라는 문법이다.

주의 할 점:
삼항 연산자를 사용하여 코드의 라인이 줄어들었다고 컴파일 속도가 빨라지는 것은 아니다.
삼항 연산자를 중복해서 처리할 경우, 가독성이 떨어질 수 있다.
